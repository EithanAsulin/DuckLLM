Documentation

DuckLLM Is An Open Source (GPL v3 License) Project Made By Eithan Asulin, The Purpose Of This Is To Be Able To Help Users Which Want To Use AI Without The Entire Sending Data Thing, This Project Unlike Others Is Purely For Privacy With 0 Cloud Dependencies & 0 Data Collection.

DuckLLM's Entire Purpose Is To Bring a Smart, Helpful AI Without The Need For Internet And Without The Worries Of Privacy All Code Can Be Viewed And Edited To Ensure User Trust.

DuckLLM Is Most Stable On Linux Due To It Being The Operating System Developed On, If You're Interested In Helping With Testing You're Free To DM @duck_ae On Discord With Your Operating System Info.

Credits For The Original Model (Qwen 2.5 3b/7b) Go To Alibaba Cloud For Their Incredibly Impressive Local Model.

Linux Installation

Unified guide for all Linux distributions.

  1. 1. Download Install DuckLLM.zip
  2. 2. Extract Extract the files in your.
  3. 3. Access Right Click ➔ Open Terminal Here, or cd into the extracted directory.
  4. 4. Execute Run the installer script: 
    python installer.py
  5. 4. Pick Your Distro Or Pick "Other" If Yours Isn't Listed
