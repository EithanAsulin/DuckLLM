# DuckLLM
OPEN-GPLv3 License (Open Source)

Release V1.0.0 STABLE (LINUX)/BETA (WINDOWS) Out In Releases
----------------------------
DuckLLM Uses phi3:mini From Ollama Allowing It To Run On Windows Even On Lower End Hardware.

Python & Ollama Is Required For The Installer To Work!

Requirements
--------------

Windows Requirements :
- Minimum -
Cpu - i5 10200U/Ryzen 1600/i5 8th Gen (Can Run On Lower End Hardware But Very Slowly)
Ram - 8gb-12gb DDr3 (Depends If Its Bloated Or Not)
GPU - Integrated Graphics Are Enough

- Recommended -
i5 12th Gen Laptop (U Series)/Ryzen 3600x/i5 8th Gen (Should Run Without Delays But Dont Try MultiTasking With more Than 5 Windows)
Ram - 12gb/16gb DDr4 (More Than Enough)
GPU - Integrated Graphics For Intel 12th gen/Radeon 7000/GTX 1660 Super

Linux Requirements :

- Minimum -
i5 5200U/Ryzen 1000
6gb DDR3 RAM
Intel UHD 4000

- Recommended -
i5 8th Gen/Ryzen 3500x
8-12GB RAM DDR4
Integrated Graphics
